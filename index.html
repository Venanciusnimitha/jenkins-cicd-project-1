🚀 Jenkins CI/CD Docker Integration Project

This project demonstrates a complete CI/CD pipeline using Jenkins, Docker, and GitHub.

🎯 Project Overview

The main objective of this project is to automate the process of building, testing, and deploying a Docker container using Jenkins. The pipeline performs the following steps:

  • ✅ Clone source code from a GitHub repository
  • ✅ Build a Docker image using a Dockerfile
  • ✅ Push the built image to Docker Hub
  • ✅ Clean up Docker resources after a successful push

📝 Project Details

GitHub Repository: https://github.com/Venanciusnimitha/jenkins-cicd-project-1

Docker Hub Image: https://hub.docker.com/r/nimitha1111/my-httpd-image

📚 Technologies Used

  • 🔹 Jenkins for CI/CD automation
  • 🔹 Docker for containerization
  • 🔹 Git & GitHub for version control

🎉 Final Outcome

The successfully built Docker image is pushed to Docker Hub and can be pulled and deployed on any environment using the following command: 

docker pull nimitha1111/my-httpd-image
docker run -d -p 8080:80 nimitha1111/my-httpd-image
